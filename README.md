# Personal Portfolio Website

Simple single-page portfolio website built with HTML and CSS.

## Overview
This project is a personal portfolio landing page with key sections:
- Hero
- About
- Skills
- Projects
- Contact

It is designed as a clean static site and can be opened directly in a browser.

## Tech Stack
- HTML5
- CSS3

## Project Structure
```text
my portfolio/
- index.html
- styles.css
- README.md
```

## Run Locally
1. Open the project folder.
2. Double-click `index.html`.

Or use VS Code Live Server:
1. Install the Live Server extension.
2. Right-click `index.html` and select `Open with Live Server`.

## Current Features
- Fixed top navigation bar
- Full-screen hero section with call-to-action button
- About section with profile image
- Skills grid cards
- Projects grid cards
- Contact form UI
- Footer section

## Notes
- Images are currently loaded from `https://picsum.photos`.
- Content text can be customized with your personal information and project details.
